# AI Engineering Masterclass

A comprehensive, interactive tutorial covering AI Engineering from fundamentals to production deployment.

## Course Overview

This masterclass covers 12 comprehensive tracks:

1. **Foundations** - Math, Python, Data Engineering
2. **Classical ML** - Supervised/Unsupervised learning, algorithms
3. **Deep Learning** - Neural networks, CNNs, RNNs
4. **Transformer Revolution** - Attention, LLMs
5. **Modern LLM Ecosystem** - GPT, Claude, open-source models
6. **Prompt Engineering** - Zero-shot, CoT, patterns
7. **RAG & Knowledge Systems** - Embeddings, vector DBs, pipelines
8. **Fine-Tuning** - LoRA, QLoRA, DPO, alignment
9. **Agentic AI** - ReAct, LangGraph, CrewAI, multi-agent
10. **AI Engineering & MLOps** - Serving, observability, cost
11. **Hardware & Infrastructure** - GPUs, distributed training
12. **Frontier & Future** - Emerging models, safety, multimodal

## Quick Start

Open `ai-engineering-masterclass.html` in any modern browser.

No server required - just open the file directly.

## Features

- **12 Tracks** with subsections and quizzes
- **Animated SVG diagrams** for all architectures
- **Interactive code playgrounds** with syntax highlighting
- **Progress tracking** with localStorage
- **Dark/Light mode toggle**
- **Search functionality** across all topics
- **Bookmarks** for key concepts
- **BFSI/Insurance use case callouts**

## Target Audience

- Software engineers transitioning to AI
- Data scientists building production ML systems
- Developers working with LLMs and agents
- Technical product managers in AI companies
